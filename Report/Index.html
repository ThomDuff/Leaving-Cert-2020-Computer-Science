1.2 Response to the Brief

  • In order to cement an idea I first identified my target audience, this being junior cert students as educating the youth is important and they will also be learning the similar stuff ("Co2", "Ph" and the climate cycle) in school, killing two birds with one stone

  • In order to choose a data set to visualise I picked the most direct and therefore easy to understand, this being the impact of rising Co2 levels and ocean acidification.

  • The Mauna Loa ocean acidifcation graph is what inspired me to use the relationship between Co2 and Ocean Ph as there was a direct scientific link

  • The target audience being junior cert students I decided to base my website off the science textbook they use "Essential Science", using the same turquoise colour scheme.

    4.1 Reflection on meeting the brief

    Personally I am delighted with the project as I fullfilled all my design ambitions, these being creating a 2 panel line chart, matching the essential science textbook colour shceme and having a comment box which pushes to and pulls from firebase

    1. The chosen dataset must be prepared, cleaned and stored in a suitable file system or database

    My program written in python analyses the dataset by reading it, skipping the header and iterating through each row. It interprets each the first and second appending each to a list, converting the second row of each file to a float along the way
